## Frontend

The frontend of this project is created with ReactJs. Redux Toolkit and Firebase is used for the authentication.

To start the frontend:
`cd client`
`npm install`
`npm run dev`

Also .env file has to be created.
.env content:
`VITE_FIREBASE_API_KEY= from firebase`
